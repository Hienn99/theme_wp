== Description ==

Integrate Javascript, JQuery, Css libraries into the website

**Features**

* Use for the front-end
* Css: Font Awesome 4.7.0, Font Awesome 5.6.1
* jQuery lib: 
		- Fancybox Version 3.5.7, 
		- OwlCarousel 2.3.4, 
		- Slick 1.8.1, 
		- Flickity 2.2.1, 
		- JQuery Confirm 3.3.4, 
		- Swiper Slider 4.5.0
		- Waypoints 2.0.3
		- Countdown 2.2.0

**Contact Us**

Web design: [hiendev](https://github.com/Hienn99)


== Installation ==

1. Upload folder `js-css-libs-wpshare247` to folder `/wp-content/plugins/`
2. Active from menu **Plugins** (**Plugins > Installed Plugins**).
